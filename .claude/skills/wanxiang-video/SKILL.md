# 万相AI生视频技能

阿里万相AI生视频是一款AI视频生成工具，可以根据文字描述和图片生成动态视频。

## 产品简介

万相AI生视频是阿里云通义万相推出的AI视频生成工具，支持：
- 文字生成视频
- 图片生成视频
- 多风格视频创作

## 基础操作步骤

### 1. 选择创作模式

- **文生视频**：输入文字描述生成视频
- **图生视频**：上传图片生成动态视频

### 2. 编写提示词

#### 优质提示词公式

```
[主体描述] + [场景环境] + [运动/动作] + [风格/氛围] + [拍摄方式]
```

#### 提示词示例

**人物类**：
```
一位年轻女子在樱花树下翩翩起舞，粉色花瓣随风飘落，阳光透过树叶洒下斑驳的光影，电影质感，缓慢推进镜头
```

**风景类**：
```
日出时分，金色的阳光洒在海面上，波光粼粼，海鸥在空中盘旋，画面从左向右缓慢平移，纪录片风格
```

**物品类**：
```
一只复古木质手表放在桌面上，齿轮缓缓转动，指针走动，特写镜头，复古电影质感
```

### 3. 调节参数

| 参数 | 说明 |
|------|------|
| 运动幅度 | 控制镜头和物体的运动强度 |
| 运镜方式 | 推、拉、摇、移、跟 |
| 氛围 | 明亮/柔和/电影感 |

### 4. 生成与下载

- 点击生成按钮
- 等待视频渲染（通常1-3分钟）
- 预览满意后下载保存

## 高级技巧

### 提示词优化

1. **使用具体词汇**：避免模糊描述，用具体名词和动词
2. **添加情感词**：如"浪漫"、"紧张"、"欢乐"等
3. **指定视角**：如"俯视"、"仰视"、"特写"
4. **添加风格词**：如"赛博朋克"、"水墨画"、"皮克斯风格"

### 运镜指令

```
- 特写镜头 (close-up shot)
- 远景镜头 (wide shot)
- 缓慢推近 (slow push-in)
- 缓慢拉远 (slow pull-out)
- 摇镜头 (pan left/right)
- 跟随镜头 (follow shot)
- 环绕运镜 (orbiting shot)
```

### 常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 视频模糊 | 升级会员使用1080P |
| 动作不自然 | 减少同时发生的动作数量 |
| 生成失败 | 精简提示词，避免复杂描述 |
| 画面撕裂 | 减少大幅度的运动描述 |

## 提示词库

### 常用场景模板

**自然风光**：
```
[季节]的[地点]，[天气]，[光线描述]，[运动描述]，[风格]
```

**人物动作**：
```
[性别/年龄]的[角色]正在[动作]，[表情]，[服装]，[环境]，[风格]
```

**产品展示**：
```
[产品名称]放置在[场景]，[特写/全景]，[光线]，[运动]，[风格]
```

### 否定提示词

如需避免某些元素，可添加否定提示词：
- 避免抖动：no camera shake, stable
- 避免变形：no deformation, no distortion
- 避免模糊：no blur, sharp focus
