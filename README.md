## 🔄 工作流程

1. **收集**：将新想法和笔记先放入 [[00inbox|收集箱]] 对应分类目录
2. **整理**：定期处理收集箱内容，分类到适当位置
3. **连接**：为笔记添加标签和链接，融入知识网络
4. **回顾**：定期回顾重要笔记，巩固记忆和更新内容
